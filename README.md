## Hey, I’m Kushan 🙋🏻‍♂️

Senior Software engineer at HSBC with a backend + platform focus. I ship governed data access experiences, Kubernetes-first delivery, and contract-first APIs. Always tuning the SDLC so teams ship fast without trading away security.

**Currently**: building on Kubernetes/Helm, supporting data teams across Hadoop/Impala/Doris/Kafka/Redis, and leveling up platform automation.

**Key skills**

- APIs: contract-first design (OpenAPI), versioned/compatible services, validation + resilient error models; Java/Spring Boot backends with TypeScript/React/Next.js frontends (Tailwind CSS)
- Languages: Java, TypeScript/JavaScript, Python, Scala, C++
- Data & streaming: governed access and audited writebacks across Hadoop, Spark/PySpark, Impala, Doris, Hive, Kafka, Redis; SQL/NoSQL stores (PostgreSQL, MySQL, MongoDB, ElasticSearch)
- Kubernetes/Helm & DevOps: Docker, Kubernetes/Helm, GitHub Actions, Jenkins, Ansible, Git; multi-env promotion, rollbacks, config separation, resource tuning for data-heavy services
- Observability & orchestration: Splunk, Airflow, Control-M, Yarn; quality/security gates (Sonar, Nexus IQ), compliance-first pipelines
- Security & reliability: zero-vuln gating, sane base images, secrets hygiene, solid authz patterns; performance/resilience tradeoffs, schema evolution, freshness vs latency
